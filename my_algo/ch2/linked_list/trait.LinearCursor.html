只读线性游标特质, 用于实现只读的循位置访问.

Required methods

fn peek(&self) -> Option<&T>

获得所指结点的元素的只读引用. 表空或指向幽灵结点时返回None.

fn is_front_or_empty(&self) -> bool

表空或所指结点为首结点时返回true. 特别地, 指向“幽灵”结点时返回false.

fn is_empty(&self) -> bool

表为空时返回true. 特别地, 若指向“幽灵”结点则返回false.

fn is_ghost(&self) -> bool

是否为“幽灵”结点.

fn index(&self) -> Option<usize>

所指结点相对与首结点的偏移. 若表空则返回None.

消耗游标, 转换为内容的只读引用.
